# 📅 EscalaFácil

Aplicação web para importar escalas de trabalho em PDF e exportar para o calendário.

## ✨ Funcionalidades

- 📎 **Importar PDF** da escala automaticamente
- ✏️ Colar texto manualmente como alternativa
- 🔤 **Tabela de turnos configurável** (já pré-preenchida com todos os códigos A–Z)
- 👤 Selecionar o nome do funcionário
- 📋 Visualizar a escala interpretada com horários completos
- 📆 Exportar para **Google Calendar**, **iPhone/iPad**, **Android** via arquivo `.ics`
- 🖨️ Imprimir ou salvar como PDF
- 🕐 Histórico de escalas importadas (salvo localmente no navegador)

## 🚀 Como usar

Basta abrir o arquivo `index.html` no navegador — não precisa instalar nada.

Para disponibilizar online para outros funcionários, hospede em:
- [GitHub Pages](https://pages.github.com/)
- [Netlify](https://netlify.com/) (arraste a pasta, pronto)
- [Vercel](https://vercel.com/)

## 📱 Como exportar para o celular

| Plataforma | Instruções |
|---|---|
| iPhone/iPad | Toque no `.ics` baixado → "Adicionar todos" |
| Android | Abra o `.ics` com o Google Calendar |
| Computador | Clique duas vezes no `.ics` |

## 🔤 Tabela de turnos padrão

A tabela já vem com todos os códigos da tabela de referência (A, B, B1, C, C1, C2... até Z).
Você pode editar, adicionar ou remover qualquer código no Passo 2 do aplicativo.
